Like Statement (Examples)


 1. Finding all employees whose name starts with "J": 

	SELECT * FROM Employees WHERE FirstName LIKE 'J%';

 2. Using _ (Underscore) to Match a Single Character: 

	SELECT * FROM Employees WHERE FirstName LIKE 'J___';

 3. Finding Names Containing a Specific Substring: 

	SELECT * FROM Employees WHERE FirstName LIKE '%an%';

 4. Finding Names Ending with a Specific Substring: 

	SELECT * FROM Employees WHERE LastName LIKE '%son';

 5. Combining LIKE with Other Conditions: 

	SELECT * FROM Employees WHERE FirstName LIKE 'J%' AND Department = 'HR';

 6. Using NOT LIKE to Exclude Patterns: 

	SELECT * FROM Employees WHERE FirstName NOT LIKE 'J%';

 7. Case Insensitivity (By Default in MSSQL): 

	SELECT * FROM Employees WHERE FirstName LIKE 'j%';
