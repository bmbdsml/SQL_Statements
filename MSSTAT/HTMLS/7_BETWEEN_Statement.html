Between Statement (Examples)


 1.  Example with Numbers: 

			Let's say you have a `Sales` table and you want to find all sales amounts between 1000 and 5000:

	 SELECT  * FROM Sales  WHERE  Amount  BETWEEN   1000  AND  5000;

   2.  Example with Dates: 

			If you want to select records between two specific dates in a `Orders` table:

	 SELECT  * FROM Orders  WHERE  OrderDate  BETWEEN  '2024-01-01' AND '2024-12-31';

 3.  Example with Strings: 

			If you're looking for records where the `ProductName` is alphabetically between certain values:

	 SELECT  * FROM Products  WHERE  ProductName  BETWEEN  'Apple' AND 'Orange';

 4.  Example with Numeric Range (Inclusive): 

			Suppose you have a `Employees` table, and you want to find employees whose salary falls between 30,000 and 60,000:
	
	 SELECT  EmployeeID, Name, Salary FROM Employees  WHERE  Salary  BETWEEN  30000 AND 60000;

 5.  Example with Date Range: 

			If you want to find all orders made between two specific dates and filter by `OrderDate`:

	 SELECT  OrderID, CustomerID, OrderDate FROM Orders  WHERE  OrderDate  BETWEEN  '2025-02-01' AND '2025-02-10';

 6.  Example with Time Range: 

			If you need to filter records based on a time range, say in a `FlightSchedules` table, 
			you could use `BETWEEN` with `TIME` values:

	 SELECT  FlightID, DepartureTime, ArrivalTime FROM FlightSchedules  WHERE  DepartureTime  BETWEEN  '06:00:00' AND '12:00:00';

 7.  Example with Date and Time (Datetime Range): 

		You could combine both date and time in a `BETWEEN` query, for instance, 
		selecting logs within a particular datetime range in a `LogEntries` table:

	 SELECT  LogID, LogMessage, LogDate FROM LogEntries  WHERE  LogDate  BETWEEN  '2025-01-01 10:00:00' AND '2025-01-31 18:00:00';

 8.  Example with Strings (Alphabetical Range): 

		If you're interested in getting products that start with certain letters, let's say between 'B' and 'M':

		This query will return all products where the `ProductName` alphabetically falls between 'B' and 'M'.

	 SELECT  ProductID, ProductName FROM Products  WHERE  ProductName  BETWEEN  'B' AND 'M';

 9.  Example with Negative Numbers: 

		If your `Balance` field can have negative values (e.g., in a bank transactions table), 
		you could use `BETWEEN` to find balances between -1000 and 0:

	 SELECT  TransactionID, Balance FROM Transactions  WHERE  Balance  BETWEEN  -1000 AND 0;

 10.  Example with `NOT BETWEEN`: 

		You can also use the `NOT BETWEEN` operator to exclude a range of values. 
	
		For example, if you want to select employees whose salary is  not  between 30,000 and 60,000:

	 SELECT  EmployeeID, Name, Salary FROM Employees  WHERE  Salary  NOT BETWEEN  30000 AND 60000;

 11.  Example with `BETWEEN` in an `ORDER BY` Clause: 

		You can use `BETWEEN` in a query with an `ORDER BY` clause to order the records within the range:

		This query will return all products priced between 10 and 100, sorted by price.

	 SELECT  ProductID, ProductName, Price FROM Products  WHERE  Price  BETWEEN  10 AND 100 ORDER BY Price;

 12.  Example with `BETWEEN` on Floating-Point Numbers: 

		If you're working with floating-point numbers, like measurements or percentages:

		This query will return products with ratings between 4.0 and 5.0.

	 SELECT  ProductID, ProductName, Rating FROM Products  WHERE  Rating  BETWEEN  4.0 AND 5.0;
