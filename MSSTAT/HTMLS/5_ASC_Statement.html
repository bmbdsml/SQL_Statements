ASC Statement (Examples)

 1. ASC -> Example-1 

	 SELECT  FirstName, LastName, Salary FROM Employees  ORDER BY  Salary  ASC ;

			-  ORDER BY  Salary : Specifies the column (`Salary`) by which the results will be sorted.
	
			-  ASC  : Sorts the results in ascending  order (from the smallest to the largest). 

		This is actually the default sorting order, so you can omit `ASC` and it will still work the same way.
		
 1. ASC -> Example-2 

				Example: Sorting Multiple Columns in Ascending Order

				You can also use `ASC` with multiple columns:

	 SELECT  FirstName, LastName, HireDate FROM Employees  ORDER BY  LastName  ASC , FirstName  ASC ;
