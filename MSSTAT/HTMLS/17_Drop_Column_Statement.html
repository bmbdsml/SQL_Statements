Drop Column Statement (Examples)

1 < Drop | Column > 

	ALTER TABLE table_name DROP COLUMN column_name;

2 < Drop | Foreign Key Constraint > 

	ALTER TABLE Orders DROP CONSTRAINT FK_Orders_Customers;

3 < Drop | Primary Key Constraint > 

	ALTER TABLE Employees DROP CONSTRAINT PK_Employees_Id;

4 < Drop | Check Constraint > 

	ALTER TABLE Employees DROP CONSTRAINT CK_Employees_Age;
