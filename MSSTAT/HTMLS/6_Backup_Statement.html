Backup Database (Examples)


 1. BACKUP DATABASE YourDatabaseName 

		 TO DISK  = 'C:\Backups\YourDatabaseName_Backup.bak'

		 WITH FORMAT,

		     MEDIANAME  = 'YourDatabaseNameBackup',

		     NAME  = 'Full Backup of YourDatabaseName';

- [ BACKUP DATABASE  YourDatabaseName] : 

		This specifies the name of the database you want to back up (`YourDatabaseName` in this example).

- [ TO DISK = 'C:\Backups\YourDatabaseName_Backup.bak'] : Specifies the location and name of the backup file. 

		In this case, the backup will be stored in the `C:\Backups\` directory with the file name `YourDatabaseName_Backup.bak`.

- [ WITH FORMAT ] : 

		Ensures that the backup file is initialized, effectively erasing any existing backup sets on the destination file
		before writing the new backup.

- [ MEDIANAME ] :

		 Specifies a name for the backup media set, which is useful for managing multiple backup files.

- [ NAME ] : 

		A description for the backup that you can identify later (e.g., `Full Backup of YourDatabaseName`).

  2. Backup with Compression (SQL Server 2008 and later) 

		If you're using SQL Server 2008 or later and you have the option for backup compression, 

		you can use the `COMPRESSION` option to reduce the size of the backup file:

	 BACKUP DATABASE  YourDatabaseName 

	  TO DISK  = 'C:\Backups\YourDatabaseName_Backup.bak'

	  WITH COMPRESSION, FORMAT, 

	     MEDIANAME  = 'YourDatabaseNameBackup',

	     NAME  = 'Compressed Full Backup of YourDatabaseName';

  3. Backup Only Specific Filegroup 

	If you have a database with multiple filegroups, you can back up a specific filegroup:

	  BACKUP DATABASE  YourDatabaseName

	  FILEGROUP  = 'Primary'

	  TO DISK  = 'C:\Backups\PrimaryFilegroup_Backup.bak'

	  WITH NAME  = 'Backup of Primary Filegroup';

  4. Backup Transaction Log 

	To back up the transaction log (useful for point-in-time recovery), you can use the following statement:

	 BACKUP LOG  YourDatabaseName

	  TO DISK  = 'C:\Backups\YourDatabaseName_Log_Backup.trn'

	  WITH NAME  = 'Transaction Log Backup';
