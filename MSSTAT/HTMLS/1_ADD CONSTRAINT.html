Add Constraint (Examples)


  1.  Adding a Primary Key Constraint 

		To add a primary key constraint to an existing column:
 
		 ALTER TABLE  Employees 

		 ADD CONSTRAINT  PK_Employees_ID  PRIMARY KEY  (EmployeeID);

   2.  Adding a Foreign Key Constraint 

		To add a foreign key constraint between two tables:
 
		 ALTER TABLE  Orders

		 ADD CONSTRAINT  FK_Orders_EmployeeID  FOREIGN KEY  (EmployeeID)

		 REFERENCES  Employees(EmployeeID);

   3.  Adding a Unique Constraint 

		To ensure that all values in a column are unique:
 
		 ALTER TABLE  Products

		 ADD CONSTRAINT  UQ_ProductCode  UNIQUE  (ProductCode);

   4.  Adding a Check Constraint 

		To add a constraint that ensures a column meets a specified condition:
 
		 ALTER TABLE  Employees

		 ADD CONSTRAINT  CHK_Age  CHECK  (Age >= 18);

   5.  Adding a Default Constraint 

		To specify a default value for a column when no value is supplied:
 
		 ALTER TABLE  Orders

		 ADD CONSTRAINT  DF_OrderDate  DEFAULT  GETDATE() FOR OrderDate;

   6.  Adding a Not Null Constraint 

		To add a `NOT NULL` constraint to a column:

		 ALTER TABLE  Employees

		 ALTER COLUMN  EmployeeName VARCHAR(100)  NOT NULL  ;

		   Example of Multiple Constraints  in One Statement:

		 ALTER TABLE  Employees
	
		 ADD CONSTRAINT  PK_EmployeeID  PRIMARY KEY  (EmployeeID),

		 CONSTRAINT  CHK_Age  CHECK  (Age >= 18),

		 CONSTRAINT  UQ_Email  UNIQUE  (Email);
