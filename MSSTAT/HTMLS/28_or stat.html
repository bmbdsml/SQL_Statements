OR Statement (Examples)


 1. OR Statement for Character Fields  

	SELECT *  FROM  Employees WHERE  Department = 'Sales'  OR  Department = 'Marketing'; 

 2. Selecting records with multiple conditions on different columns 

	 SELECT  *  FROM  Employees  WHERE  Department = 'HR'  OR  Salary > 70000;

 3. Using OR with NULL values 

	 SELECT  *  FROM  Products  WHERE  Discount  IS NULL OR  Discount = 20; 

 4. Combining OR with AND 

	 SELECT  *  FROM  Employees  WHERE  (Department = 'Sales'  OR  YearsOfExperience > 10)  AND  Salary > 50000;

 5. Filtering based on multiple values in a column 

	 SELECT  *  FROM  Customers  WHERE  City = 'New York'  OR  City = 'Los Angeles'  OR  City = 'Chicago';

 6. Using OR with different data types (e.g., dates) 

	 SELECT  *  FROM  Orders  WHERE  OrderDate = '2024-01-01'  OR  OrderDate = '2024-02-10';

 7. Using OR for a range of values 

	 SELECT  *  FROM  Products  WHERE  CategoryID = 1  OR  CategoryID = 2  OR  CategoryID = 5;

 8. Using OR with LIKE (for pattern matching) 

	 SELECT  *  FROM  Customers  WHERE  Name  LIKE  'A%'  OR  Name  LIKE  'B%';
