Drop Table Statement (Examples)


1 < Drop | Table  > 

	DROP TABLE Employees;

2 < Drop | IF Table Exist > 

	DROP TABLE IF EXISTS Employees;

3 < Drop | Multiple Table  > 

	DROP TABLE Orders, Products, Employees;

4 < Drop | Specific Schema > 

	DROP TABLE Sales.Transactions;

5 < Drop | If Foreign Key Exist for table to be Dropped > 

	ALTER TABLE Orders

	DROP CONSTRAINT  FK_Orders_Customers;

	DROP TABLE  Customers;

6 < Drop | Dynamic SQL > 

	DECLARE @TableName NVARCHAR(128) = 'Products';

	EXEC('DROP TABLE ' + @TableName);
